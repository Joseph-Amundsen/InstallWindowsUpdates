<h1>Installing Windows Updates</h1>

<h2>Description</h2>
This project provides a Standard Operating Procedure (SOP) for guiding individuals through the process of checking for and installing Windows updates. This is important for a security standpoint so every computer has the latest patches to protect against new, known vulnerabilities.
<br />


<h2>Objectives</h2>

- <b>Provide clear, step-by-step instructions for checking for and installing Windows updates.</b>
- <b>Ensure devices receive the latest security patches, bug fixes, and feature updates.</b>
- <b>Promote system security, stability, and optimal performance through regular updates.</b>
- <b>Create a standardized process that can be used for training, onboarding, and IT support.</b>

<h2>Intended Audience</h2>

- <b>Windows end users</b>
- <b>IT support specialists</b>
- <b>Help desk technicians</b>
- <b>System administrators</b>
- <b>New employees during onboarding</b>
